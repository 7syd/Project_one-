# Bike Sharing Demand Analysis Using Excel

## Overview
This project focuses on analyzing bike-sharing demand using Microsoft Excel. The goal was to study how factors like weather, temperature, time, holidays, and user type affect bike rentals. The project also includes dashboard creation, forecasting, anomaly detection, and VBA automation.

---

# Features

- Data cleaning and preprocessing
- Pivot table analysis
- Interactive dashboard
- KPI cards
- Slicers and timeline filters
- Forecasting using Excel Forecast Sheet
- Anomaly detection using statistical methods
- VBA automation for dashboard refresh and cleaning

---

# Dashboard Insights

- Peak rentals mostly occur during commute hours.
- Registered users dominate bike rentals.
- Weather conditions impact rental demand.
- Moderate temperatures show higher rental activity.
- Forecasting indicates stable future demand trends.
